# StrangerTEC-GP-GR
Proyecto de Fundamentos de Sistemas Computacionales

## Como usar?
### Clonar el repositorio:
```gitattributes
git clone https://github.com/unmolten/StrangerTEC-GP-GR
```
### Descargar la libreria requests para python:
```
pip install requests
```
### Configuracion:
* En ```rpi-main.py``` ingresa el nombre de la red y la contraseña respectivamente en:
  
  ```python
  # credenciales de red
  ssid ='(aqui)'
  password ='(aquí)'
  ```
### Correr el codigo:
* Correr ```rpi-main.py``` en Thonny con MicroPython
* Correr ```StrangerTEC-GUI.py``` en un IDE con un interpretador de Python.
